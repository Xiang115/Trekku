# Trekku UI Prototype

Frontend UI prototype for the Trekku travel e-commerce project.

## Features

- AI trip planner interface
- Persona-based package selection
- Login and sign up prototype flow
- Dynamic budget customization
- AI itinerary dashboard
- Shared planning workspace
- Monthly insight dashboard
- Savings comparison checkout

## Run Locally

From this folder:

```powershell
python -m http.server 4173
```

Then open:

```text
http://localhost:4173
```
