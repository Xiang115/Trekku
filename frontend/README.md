# Trekku Frontend

React + Vite (TypeScript) single-page app for the Trekku travel platform, wired to the FastAPI backend.

## Features

- **Chat planner** — conversational trip planning via `POST /agent/chat` (multi-turn, session persisted in `localStorage`).
- **AI itinerary** — day-by-day hotel / attraction / flight cards generated by the agent, with:
  - star ratings → `POST /agent/feedback`
  - entity swap (pick an alternative from `/ratings/entities`) → `POST /agent/modify`
- **Insights** — real price / rating / review trends from `/ratings/cities`, `/ratings/entities`, `/ratings/trend/...`.
- **Savings checkout** — mock bundle comparison UI (no payment backend).
- **Demo auth** — local-only login/signup stored in `localStorage` (the backend enforces no auth).

The visual theme (colors, layout, components) is preserved from the original prototype via `src/styles/theme.css`.

## Prerequisites

- Node 18+ (developed on Node 22)
- The Trekku backend running on `http://localhost:8000` (see `../backend`).

## Setup

```powershell
npm install
copy .env.example .env   # optional; leave VITE_API_BASE_URL empty to use the dev proxy
```

`VITE_API_BASE_URL` is empty by default, so API calls go same-origin and the Vite dev
proxy (in `vite.config.ts`) forwards `/agent`, `/ratings` and `/health` to `localhost:8000`.
Set it to an absolute URL for production builds.

## Run

```powershell
# 1) Backend (from ../backend)
.\.venv\Scripts\python.exe -m uvicorn main:app --host 127.0.0.1 --port 8000

# 2) Frontend (from this folder)
npm run dev        # http://localhost:5173
```

## Build

```powershell
npm run build      # type-checks (tsc -b) then bundles to dist/
npm run preview    # serve the production build
```

## Project structure

```
src/
  api/        typed client (client.ts) + backend types (types.ts)
  components/ planner/ · itinerary/ · insights/ · checkout/ + Topbar, AuthModal, AccountSection
  state/      AuthContext (demo auth) · ChatContext (session, conversation, itinerary)
  hooks/      useRatings (cities)
  icons/      inline SVG sprite + <Icon> wrapper
  lib/        format.ts (MYR currency / compact / date helpers)
  styles/     theme.css (original theme, verbatim) + additions.css (new UI)
  config/     env.ts (API base URL)
```
